#### 3.1.4 业务流程
| 正常流程 | 1.学生向系统发起导出课程表的请求；                                                                                                    2.系统向学生提供一个输入教务处账号和密码的登录框；                                                                                  3.学生输入账号和密码；                                                                                                                           4.系统将课程导入，并跳转到课程表栏目下显示课程表； |
| -------- | ------------------------------------------------------------ |
| 扩展流程 | 2a、学生输入的账号或用户名不正确；                                                                                                 1、系统提示登录失败；                                                                                                                           2、学生重新输入账号和用户名；                                                                                                               重复2a操作，直至登录成功，返回正常流程4。 |
